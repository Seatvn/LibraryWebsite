Library Website

Это простой веб-сайт для библиотеки, созданный на Java с использованием Spark Framework и базы данных SQL Server. Сайт позволяет управлять книгами: добавлять, удалять, изменять данные и просматривать список книг.

Основные возможности:
Просмотр списка всех книг в библиотеке,
Добавление новых книг,
Редактирование информации о книгах,
Удаление книг,
Подключение к базе данных SQL Server через JDBC.

Технологии:
Java 17+,
Spark Framework,
JDBC,
SQL Server.
