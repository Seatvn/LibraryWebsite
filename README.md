# Описание проекта

Это простой веб-сайт для библиотеки, созданный на Java с использованием Spark Framework и базы данных SQL Server. Сайт позволяет управлять книгами: добавлять в избранное, добавить в прочитанное и просматривать список книг.

# Основные возможности

Просмотр списка всех книг в библиотеке

Добавление книг в избранное и прочитанное

Подключение к базе данных SQL Server через JDBC

# Технологии

Java 17+

Spark Framework

JDBC 

SQL Server

# Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/seatvn/LibraryWebsite.git

```
Если вы не используете Git, то вы можете просто скачать исходный код репозитория в ZIP-архиве и распаковать его на свой компьютер.

2. Откройте проект в любимой IDE (IntelliJ IDEA, Eclipse и др.)
   
4. Убедитесь, что SQL Server запущен и база данных LibraryDB создана.
   
6. Запустите TestDB.java для проверки подключения к базе.
   
8. Запустите основной сервер приложения (Main.java).
